Acidpop Telegram Basic BOT Help

사용방법 : Command [Option]

Command List
	사진
		- 라즈베리파이 카메라로 사진 촬영 후 전송
	날씨 <동네>
		- 동네 예보 조회
	토렌트 <키워드>
		- 토렌트 검색 후 검색 된 내용 출력
	다운 <Index>
		- 토렌트 명령어로 검색된 목록 중 Index 에 해당되는 토렌트를 다운로드한다.
	목록
		- 검색된 토렌트 목록을 다시 출력한다
	wol <Index>
		- 등록된 WOL 시스템 중 Index 에 해당되는 시스템에 WOL 패킷을 전송한다.
	저항 <띄 색깔>
		- 검(검정), 적(빨강), 주(주황), 노(노랑), 초(초록), 파(파랑), 
		  보(보라), 회(회색), 흰(흰색), 금(금색), 은(은색), 무(없음)
	서버 <켜> <꺼> <상태> <강제종료>
		- 켜줘/켜/켜기/켜짐/켜라/켜줘라/on
		- 꺼줘/꺼/끄기/꺼짐/꺼라/꺼줘라/off/shutdown/꺼주세요
		- 강제종료 : 전원버튼을 4초간 누르는 상태로 만듬
		- 상태/지금/어떤데/켜졌냐?/꺼졌냐?/켜졌냐/꺼졌냐
	상태
		- 현재 IP, 라즈베리의 CPU의 온도와 사용률, 메모리와 디스크의 사용량, 로그온상태를 알려준다.
	쉘모드
		- Shell command 모드 전환, SSH로 접속해서 사용하는 명령어 지원
	쉘모드종료
		- 쉘모드를 종료한다.
